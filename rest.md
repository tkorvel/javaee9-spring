###HTTP codes family
- 1xx - processing in progress
- 2xx - success
- 3xx - redirects
- 4xx - client's errors
- 5xx - server's errors

###Http exchange message consists of:
- request
  - url
  - headers
  - body
- response
  - status code
  - headers
  - body
